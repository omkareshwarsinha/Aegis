## ®️ What is AEGIS JARVIS

**AEGIS JARVIS v15** is a fully offline-capable **Android AI companion** designed and developed by **Omkareshwar Sinha** using the **DroidScript** platform. Unlike cloud-dependent assistants, **AEGIS** works entirely on your Android device using **local LLMs via Ollama**, while also supporting **9 online AI providers** for enhanced capabilities.

> **AEGIS** stands for **Advanced Electronic Guardian and Intelligence System** — your personal **Jarvis-like AI assistant** that runs natively on Android 9+ devices through **DroidScript's Chrome 67 WebView engine**.

**Key Highlights:**
- 🏠 **Fully Offline AI** — Run local LLMs without internet
- 🗣️ **Hinglish First** — Talk naturally in Hindi-English mix
- 📱 **40+ Built-in Apps** — From games to OSINT tools
- 🔒 **Privacy Focused** — Local encryption, no data leakage most data remain on your phone , no database , no data collection 
- ⚡ **Native Android Control** — Calls, SMS, torch, alarms, WhatsApp

## 🏗️ Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                      USER INTERACTION                        │
│              Voice Input / Text / Touch Gestures             │
└─────────────────────────┬───────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│              COMMAND LAYER — dashProcess Engine              │
│    Hinglish NLP → Intent Detection → Command Router         │
└─────────────────────────┬───────────────────────────────────┘
                             │  |
            ┌─────────────┴─────────────┐
            ▼                           ▼
┌──────────────────────┐    ┌──────────────────────────────┐
│   Native Bridge          │    │     AI ENGINE (Multi-LLM)    │
│   console.log API        │    │  Ollama / Gemini / GPT /     │
│   CMD:action:value       │    │  Groq / Grok / OpenRouter    │
└──────────┬───────────┘    └──────────────┬───────────────┘
             │                                    │
             ▼                                   ▼
┌─────────────────────────────────────────────────────────────┐
│              Aegis.js — DroidScript Native Bridge          │
│   app.TextToSpeech · app.SendIntent · app.LaunchApp        │
│   TTS hi_IN · Torch · Call · SMS · WhatsApp · Alarm        │
│   File System · Media · Contacts · Battery · Network       │
└─────────────────────────────────────────────────────────────┘
```

**AEGIS** uses a unique **console.log bridge pattern** where the HTML frontend communicates with the **DroidScript** backend through intercepted log messages, enabling seamless native Android functionality without leaving the web context.

## 🚀 Installation Guide

### Method 1: DroidScript (Recommended for Full Native Access)

```bash
# Step 1: Install DroidScript from Google Play Store
# Step 2: Create folder structure
/sdcard/DroidScript/Aegis/
├── Aegis.html          ← Main application (36,000+ lines)
├── Aegis.js            ← DroidScript native bridge
└── AEGIS_Modelfile     ← Ollama model definition

# Step 3: Open DroidScript → Load Project → Run Aegis.js
```

### Method 2: Browser Preview (AI Chat Only)

Open `Aegis.html` directly in any modern browser. **Note:** Native Android features (calls, SMS, torch) will be simulated, but AI chat functionality works fully.

### Method 3: Ollama Local AI Setup (For Offline Mode)

```bash
# On your PC (same WiFi network as Android device)
ollama pull qwen2.5:7b-instruct
ollama create aegis-jarvis -f AEGIS_Modelfile
ollama serve --host 0.0.0.0

# In AEGIS app:
# Settings → AI Provider → Ollama
# Model: aegis-jarvis
# URL: http://YOUR_PC_IP:11434
```

## 🛠️ Developer Notes

### For Contributors & Developers

**AEGIS** is built as a **single-file HTML application** for maximum portability within **DroidScript's AppType HTML** environment. The architecture follows these principles:

1. **Single File Delivery** — `Aegis.html` contains all CSS, JS, and HTML (36,000+ lines)
2. **Bridge Pattern** — Native Android access via `console.log('CMD:action:value')`
3. **ES5 Strict** — Full compatibility with Chrome 67 WebView on Android 9
4. **No Build Step** — Direct file copy deployment, no webpack or bundler
5. **Progressive Enhancement** — Works in browser (limited) and DroidScript (full)

### DroidScript Bridge API Reference

```javascript
// Text to Speech (Hindi / English)
console.log('SPEAK:Namaste, main AEGIS hoon');

// Launch Android Application
console.log('CMD:launchapp:com.whatsapp');

// Make Phone Call
console.log('CMD:call:9876543210');

// Toggle Flashlight
console.log('CMD:torch:on');
console.log('CMD:torch:off');

// Send SMS
console.log('CMD:sms:9876543210:Message body here');

// Set Alarm
console.log('CMD:alarm:07:00');

// Take Screenshot
console.log('CMD:screenshot');

// Vibrate Device
console.log('CMD:vibrate:500');
```
